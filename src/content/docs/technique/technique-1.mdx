*Technique Foundation 1 · before any reading*

<Callout bordered={true} accentColor="gold" label="Your goal for this level" icon="target">
  Sit relaxed at the correct height with level wrists, and learn to make sound by dropping a loose arm into the key instead of pressing with the fingers. No notation or sight-reading yet—just a calm, tension-free body.
</Callout>

This guide puts **technique before sight-reading**. Before a single note is read, you build the relaxed physical habit that everything else depends on. Almost every self-taught stall traces back to tension that set in early, so this is where you start.
